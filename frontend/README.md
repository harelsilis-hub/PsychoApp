# PsychoApp Frontend

React + Vite frontend for the Vocabulary SaaS application with adaptive learning.

## Tech Stack

- **Framework:** React 18
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Animation:** Framer Motion
- **Icons:** Lucide React
- **HTTP Client:** Axios
- **Routing:** React Router DOM

## Getting Started

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

The app will be available at http://localhost:5173

### Build
```bash
npm run build
```

## API Proxy

All `/api/*` requests are proxied to `http://localhost:8000/api`
